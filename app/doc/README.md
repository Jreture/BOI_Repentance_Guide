## BOI_repentance_guide
The Binding Of Isaac 한국 유저들을 위한 한글 가이드

## 소개
이것은 The Binding Of Isaac의 정보를 한글로 보고싶어하는 사람들을 위한 안드로이드 애플리케이션입니다.

- The Binding Of Isaac의 모든 정보를 제공합니다.

## 개발 환경
●안드로이드 스튜디오 버젼 : Arctic Fox 2020.3.1
● 사용 언어 : Java
● SDK 버전 : 32
● minSDK 버전 : 21
